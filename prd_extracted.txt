1. Backend Overview
This document defines the complete backend specification for OnCall Maestro — the Node.js/TypeScript server that powers alert ingestion, AI routing, escalation timers, voice handover, and rota management. This is the engineering source of truth for every developer working on the backend.
2. Project Structure
The backend follows a layered architecture. API routes handle HTTP concerns only. Services contain all business logic. Workers run async jobs. This separation means each layer can be tested and changed independently.

4. Environment Variables
All secrets are passed via environment variables. Never hardcode any key in source code. Copy .env.example to .env and fill in real values.
